# 2026-03-18 资产管理系统第一期 UAT 清单

## 1. 验收基线

本清单对应以下文档中的“第一期最小闭环”范围：

- `docs/plans/2026-03-18-asset-management-system-phase1-plan.md`

## 3. 核心验收清单

| 编号 | 场景 | 关键操作 | 预期结果 |
| --- | --- | --- | --- |
| UAT-01 | 台账建账 | 在资产台账页新增固定资产 | 资产编号由后端生成；分类、部门、责任人通过选择器录入；保存后台账可查询 |
| UAT-02 | 使用交接 | 发起领用 / 调拨 / 退还交接单 | 不允许直接改台账；交接单确认后回写使用部门、责任人、位置等台账字段，并产生变更日志 |
| UAT-03 | 盘点任务发起 | 在资产盘点页创建盘点任务并选择资产 | 仅允许选择在册 / 使用中 / 闲置 / 盘点中的固定资产；任务创建后资产进入盘点流程 |
| UAT-04 | 正常盘点回写 | 在盘点执行页登记“正常”结果 | 任务进度更新；资产状态回写为在用或闲置；最近盘点日期更新 |
| UAT-05 | 异常盘点转处置 | 在盘点执行页登记“盘亏 / 缺失 / 毁损”并选择“发起处置” | 资产状态进入 `PENDING_DISPOSAL`；盘点结果与后续动作留痕 |
| UAT-06 | 处置确认终态 | 在资产处置页确认处置 | 仅待处置资产可确认；确认后生成处置记录，资产状态进入 `DISPOSED`，且不能恢复正常使用 |
| UAT-07 | 管理统计 | 打开资产台账页查看统计卡片 | 至少能看到总量、在用、闲置、盘点中、待处置、逾期待盘点等基础统计 |
| UAT-08 | 财务导出 | 在资产台账页按当前筛选条件导出台账 | 导出字段与页面筛选口径一致，满足财务基础台账查看 |
| UAT-09 | 菜单与动态路由 | 菜单进入资产页，直接访问隐藏路由表单页 | 菜单只展示主入口；新增/编辑/详情/执行页可通过动态路由访问但不出现在菜单栏 |

## 4. 2026-03-20 资产管理员点测记录

### 4.1 点测目标

验证“盘点异常 -> 待处置 -> 处置确认 -> 台账终态”这条第一期核心闭环是否真实可走通。

### 4.2 点测环境

- 前端地址：`http://127.0.0.1:5173`
- 后端地址：`http://127.0.0.1:8080`
- 角色口径：资产管理员

### 4.3 点测样本

- 资产编号：`FA-2026-0003`
- 盘点任务号：`IV-2026-0006`
- 处置单号：`DP-2026-0002`

### 4.4 点测步骤与结果

1. 在“发起盘点任务”页创建任务 `资产管理员点测-盘点转处置-20260320`，选择资产 `FA-2026-0003`。
2. 系统成功生成任务 `IV-2026-0006`，资产进入盘点执行页。
3. 在盘点执行页对 `FA-2026-0003` 登记结果“毁损”，后续动作选择“发起处置”。
4. 提交后任务进度从 `0 / 1` 变为 `1 / 1`，资产状态变为 `PENDING_DISPOSAL`。
5. 进入资产处置页，在待处置资产池中看到 `FA-2026-0003`，执行“确认处置”。
6. 系统成功生成处置记录 `DP-2026-0002`，处置状态为“已确认”。
7. 回到资产台账页，确认 `FA-2026-0003` 的资产状态已变为 `DISPOSED`。

### 4.5 点测结论

上述链路在当前环境中已真实走通，说明第一期最关键的“盘点异常进入处置并回写终态”闭环可用。

## 5. 验收结论口径

若 `UAT-01` 至 `UAT-09` 均通过，则可认定：

- 固定资产第一期最小闭环已达到试运行条件
- 可进入试运行、问题收口与第二期规划阶段

若未通过，应优先按以下顺序整改：

1. 状态回写错误
2. 单据留痕缺失
3. 盘点 / 处置流程无法走通
4. 导出与统计口径不一致
5. 页面路由与权限配置异常
