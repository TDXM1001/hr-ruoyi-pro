# 2026-03-22 不动产占用状态矩阵第四批设计

## 1. 目标
在第三批基础上继续强化占用页签的资产管理体验，补齐三项能力：
- 自定义时间范围
- 轨迹排序切换
- 摘要卡片差异高亮

本批仍限定为前端增强，不新增 SQL、不修改后端接口。

## 2. 业务问题
当前占用页签已经能做状态矩阵、关键字筛选、时间快捷筛选，但资产管理员在复盘历史轨迹时还有三个痛点：
- 快捷时间筛选只覆盖固定窗口，无法按任意起止时间精确查轨迹
- 轨迹默认倒序，缺少正序回放能力
- 摘要卡片虽有信息，但“哪些字段发生变化”仍需要靠人工比对

## 3. 方案
采用“页签内增强”方案：
- 在轨迹工具栏增加自定义开始/结束日期与应用/清空操作
- 在轨迹工具栏增加排序切换：最新优先 / 最早优先
- 在主档联动摘要与最近一次变更摘要中加入差异高亮样式和状态标签

## 4. 详细设计
### 4.1 自定义时间范围
- 保留现有快捷时间按钮：全部时间 / 近7天 / 近30天 / 近90天
- 新增自定义时间区间：
  - 开始日期
  - 结束日期
  - 应用区间
  - 清空区间
- 规则：
  - 应用后进入 `CUSTOM` 模式
  - 快捷时间与自定义区间互斥
  - 若只填单边日期，不应用筛选
- 时间取值口径：
  - 已释放记录取 `endDate`
  - 有效占用取 `startDate`

### 4.2 轨迹排序切换
- 默认：最新优先
- 可切换：最早优先
- 排序只影响轨迹列表和“最近释放记录”判定逻辑
- 与状态/时间/关键字筛选叠加生效

### 4.3 摘要卡片差异高亮
#### 主档联动摘要
- 对比主档当前快照与当前有效占用：
  - 使用部门
  - 责任人
  - 使用位置
- 差异项显示：
  - 高亮边框
  - `待校正` 标签
- 一致项显示：
  - 正常态
  - `已同步` 标签

#### 最近一次变更摘要
- 对比上一条已释放记录与当前有效占用：
  - 部门
  - 责任人
  - 位置
- 变化项显示：
  - 高亮边框
  - `已变更` 标签
- 未变化项显示：
  - 普通态
  - `未变更` 标签

## 5. 测试设计
- 单测覆盖：
  - 自定义时间范围应用后正确过滤轨迹
  - 排序切换后列表顺序变化
  - 主档联动摘要存在差异高亮
  - 最近一次变更摘要存在差异高亮
- 浏览器点测覆盖：
  - 手动设置自定义时间区间
  - 切换排序
  - 观察摘要卡片高亮
